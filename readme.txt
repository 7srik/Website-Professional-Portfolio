# Sri Krishna's Portfolio Website

This is the official portfolio website of Sri Krishna Karumanchi. The website showcases academic details, work experience, and contact information.

##HTML5 Tags Used

main.html

1. `<meta>`: Specifies the document's character set, compatibility mode, and viewport settings.
2. `<title>`: Sets the title of the webpage displayed in the browser tab.
3. `<link>`: Links external resources such as the favicon and CSS stylesheet.
4. `<head>`: Contains meta-information about the document.
5. `<body>`: Represents the main content of the webpage.
6. `<header>`: Defines the header section of the webpage.
7. `<nav>`: Contains navigation links.
8. `<img>`: Embeds images into the webpage.
9. `<ul>`, `<li>`, `<a>`: Creates an unordered list for navigation links.
10. `<h1>`: Defines the main heading of the webpage.
11. `<p>`: Represents paragraphs of text.
12. `<i>`: Italicizes text.
13. `<button>`: Creates an interactive button for actions.
14. `<footer>`: Contains footer information.

edu.html

1. `<h2>`: Defines headings.
2. `<script>`: Embeds JavaScript for dynamic behavior.
3. `<ion-icon>`: Embeds icons from the Ionicons library.

work.html

1. `<div>`: Defines a division or section in the HTML document.
2. `<table>`: Creates an HTML table.
3. `<tr>`, `<td>`: Define table rows and data cells.

Skills.html

1. `<menu>`: Represents a list of commands or options.

contact.html

1. `<script>`: Embeds JavaScript for dynamic behavior.
2. `<details>`: Creates a disclosure widget that can be toggled open or closed.
3. `<summary>`: Defines a summary or label for a `<details>` element.
4. `<form>`: Represents an HTML form for user input.
5. `<label>`: Defines a label for an input element.
6. `<input>`: Represents an input field that users can interact with.
7. `<textarea>`: Defines a multi-line input field for text.
8. `<ion-icon>`: Embeds icons from the Ionicons library.

style.css

This CSS file contains styles for Sri Krishna's portfolio website, providing layout, colors, and positioning for various elements. It also includes responsive design adjustments for different screen sizes.

## CSS Rules and Descriptions

1. `*`: Applies styles to all elements on the page. It resets margins, padding, font-family, and sets a background color.
2. `body`: Styles for the body element, setting margin at the bottom.
3. `.maincontent`: Styles for the main content section, including background color.
4. `.maincontent img`: Styles for images inside the main content section, defining their width.
5. `nav`: Styles for the navigation bar, including flex display and alignment.
6. `nav ul li`: Styles for list items in the navigation bar, displaying them inline.
7. `nav ul li a`: Styles for navigation links, setting text color, font size, and padding.
8. `nav ul li a:hover`: Styles for navigation links on hover, changing text and background color.
9. `.detail`: Styles for the detail section, defining margins and alignment.
10. `.detail h1`: Styles for heading elements within the detail section, including font size and color.
11. `span`: Styles for the `span` element, setting text color.
12. `.detail p`: Styles for paragraphs within the detail section, defining text color and line height.
13. `.detail a`: Styles for anchor links within the detail section, specifying background, text color, and padding.
14. `.display`: Styles for a display section with positioning and margins.
15. `.display img`: Styles for images within the display section, defining their height and position.
16. `.git`: Styles for GitHub links, including font size and color.
17. `.contact`: Styles for contact information, specifying margins and alignment.
18. `.contact .phone`, `.contact .mail`, `.contact .fb`, `.contact .ln`, `.contact .loc`: Styles for contact options, including font size and color.
19. `.exp`: Styles for the experience section, setting text alignment and width.
20. `.exp .sidehead1`: Styles for subheadings within the experience section.
21. `table`: Styles for HTML tables.
22. `table td`: Styles for table cells.
23. `footer`: Styles for the footer, defining background color, text color, and position.
24. `.zoom`: Styles for elements with a zoom effect on hover, including padding and transformation.
25. `menu`: Styles for a menu element, specifying text alignment and margin.
26. `audio`: Styles for audio elements, defining their width and height.
27. `.imgpl img`, `.imgtf img`: Styles for images within specific sections, setting margin, width, and height.
28. `.imgs`: Styles for image sections, defining their display, alignment, and grid layout.
29. `.imgs .imgpl`, `.imgs .imgtf`: Styles for sub-sections within the image section, specifying float and clear properties.
30. Media queries for responsive design, adjusting styles for different screen sizes.
31. `.columnl`: Styles for a two-column layout in the education section.
